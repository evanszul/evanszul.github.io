Membuat Supaya Tampilan Bootstrap Yang Rapi Disemua Perangkat

25 Apr, 2018 · Read in about 1 min · (56 Words)
html bootsrap

Jika teman-teman selesai membuat halaman html pada komputer menggunakan bootsrap, setelah di coba di smartphone tampilannya menjadi berantakan di pc. Teman-teman dapat menggunakan kode dibawah ini untuk merapikannya. Letakkan kode dibawah ini didalam pada halaman html. 

1

<meta name="viewport" content="width=device-width,initial-scale=1">

Nah, jika sudah memasukkan kode diatas kedalam maka tampilan html yang kamu buat menjadi tidak berantakan.
